NetMatch - The End

NetMatch - The End is a multiplayer online DeathMatch-game.
There are weapons available with differences in characteristics: Double hand guns, a sub-machine gun (SMG), bazooka, shotgun, grenade launcher and chainsaw.
You can collect extra ammo and medical kits from the field to be more efficient on the battle field.
Only the pistols carry infinite ammunition.

The ranking of players is based on kill/death -ratio. Plenty of kills and few deaths will place the player on a higher position in the statistics.

In addition to human players, computer controlled AI-characters are found on the field.

Settings

Language selection

NetMatch - The End supports two in-game languages: english and finnish. The language can be selected under SETTINGS - LANGUAGE.

Game statistics

Statistics show the players kills, deaths and kill/death -ratio during gameplay.
The feature can be switched on or off under SETTINGS - GAME SETTINGS - SHOW STATISTICS or during gameplay by pressing the G-button.

SFX volume

The volume of sound effects can be adjusted under SETTINGS - GAME SETTINGS - VOLUME or during gameplay by pressing down and holding the V-button and pressing the arrow keys up (increase volume) or down (decrease volume).

Music volume

The volume of background music can be adjusted under SETTINGS - GAME SETTINGS - VOLUME or during gameplay by pressing down and holding the M-button and pressing the arrow keys up (increase volume) or down (decrease volume).

Server

Naming the server

The server needs a name so that players can identify it and join the game. The name can be anything you like.

Port

To be able to work the server needs a port to the Internet.

The port is a numeric value between 1 and 65535. However, it is recommended to use a higher value than 1024 for ports under that value may be occupied for something else.

You can only have one server per port.

Match time

The length of a match must be set to a server. The time is set in minutes.
If round changes are not wanted, enter zero (0) into this space. In this case the match will continue an unlimited time.
After each match the players are ranked.

Map selection

A map must be chosen for the server, on which the NetMatch-players play.
In addition to map name a preview of the current map is seen in the selection.

Registration and execution of the server

Registration and execution of the server make it visible for all NetMatch-players in server listing.

Execution of the server without registration

Execution without registration makes the server a hidden one, so it can only be joined by knowing it's IP-address and port.

Playing on a server

Please notice that a program running a server cannot be used to join a game because it's focusing all power to running the server.
While the server is running the screen is blue.
If the same computer that is running a server is wished to be used to join a game, another program must be launched by double clicking the game's exe-file.
However, running two programs simultaneously may cause unstability in the server program, especially shown as lag.

Controlling the server in game

Server can be controlled by using chat commands. Commands take the form of /<command> [parameters].
Underneath you will find all of the available commands. Compulsory parameters are listed like <this>
and optional parameters are listed like [this].

/admin <password> Autentikoi pelaajan hallinnoijaksi.
/kick <player> [reason] Kicks player <player> from the server.
/team <player> <team> Moves player <player> to team <team>. <team> can be 1 or Green thus moving the player to the green team tai or it can be 2 or Red thus moving the player to the red team.
/radars [ON|OFF|0|1] Toggles the radar arrows. When radar is toggled off in Team Deathmatch mode, only enemies' radars are hidden.
/timelimit <min> or /roundlength <min> Resets the map timer to start ticking from minutes <min>.
/bot limit [quantity] Sets the amount of spots for bots to fill.
/bot freeze [toggleOn] Toggles brain activity of bots. If parameter [toggleOn] is given a value other than 0 then bots are frozen.

Playing

Nick

Each player needs a freely chooseable nick in order to join the game.

Searching for servers

The server listing is searched for joinable NetMatch-servers.

Address

A server can also be joined to directly, assuming it's running and e.g. unregistered. If so, the servers IP-address must be known (see also Port).

Port

A server can also be joined to directly, assuming it's running and e.g. unregistered. If so, the servers port must be known (see also Address).

Starting the game

When the IP-address and port are known, the server is joined directly.

Playing

NetMatch - The End supports two styles of controlling the character: mouse or keyboard control.

There are two options available for character control.
This setting effects on how the character moves when using keys WASD.
The style can be selected under SETTINGS - GAME SETTINGS - CHARACTER CONTROL.

In mouse control the cursor is replaced with cross-hairs in the direction the character is facing. Left mouse button fires the current weapon and the right button toggles weapons. There's a small delay when swapping weapons.

In keyboard control the character is moved and turned with ARROW KEYS. SPACE fires the current weapon and TAB toggles weapons, again with a small delay.

Please notice that weapons can't be swapped immediately after firing but after a time the current weapon takes to reload. Reload time varies with each weapon. The SMG reloads quickly, the bazooka slowly.

It's possible to send messages to other players during gameplay. The messaging console comes out by pressing the C-button. The message written to the console is sent by pressing ENTER and exiting console by pressing ESC.

If you wish you can bring out meters showing the performance of the game by pressing the F-button. The first figure illustrates FPS (Frames Per Second) and the second figure illustrates speed of network traffic (updates per second).

Music and Maps

Music

The user can play his/her own music during gameplay by placing desired music files into the game's "music"-folder.
There can be several songs and the program will draw the next song after the previous has ended.
Supported formats are asf, it, mid, mo3, mod, mp2, mp3, ogg, raw, rmi, s3m, wav, wma and xm.

Maps

Maps used by NetMatch - The End are in the game's "map"-folder. More maps can be downloaded to the map folder from the website of NetMatch. The maps will be available when creating a server or joining a server that uses the same map.
